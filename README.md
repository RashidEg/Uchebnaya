Команды

| Команда - Ответ бота |
| /start - Начало общения |
| /name: Имя - Представиться |
| /stop - Завершить диалог |
| /numbers: 4, 3 - Передать числа |
| /set: x = 100 - Сохранить переменную |
| /undo - Удалить последний обмен |

После /numbers введите действие: +, -, *, /, gcd, avg, min, max

Форматы чисел
- Запятые: 7, 9, 12
- Пробелы: `7 9 12`
- Точка с запятой: 7;9;12
- Диапазон: 10..15 → [10,11,12,13,14,15]
- Смешанный: 5, 10..15, 20

Переменные

Последний результат сохраняется автоматически в ans:
/numbers: ans, 2
