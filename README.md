# Чат-бот

Простой чат-бот на клиентском JavaScript, который выполняет математические операции и выводит результат в окно чата.

## Запуск

Серверная часть не требуется — проект работает полностью в браузере.

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/<ваш-логин>/chat-bot.git
   cd chat-bot
   ```

2. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge, Safari).

> Просто дважды кликните на `index.html` или откройте его через меню браузера: `Файл → Открыть файл`.

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Начало общения |
| `/name: Имя` | Представиться боту |
| `/stop` | Завершить диалог |
| `/numbers: 2, 3` | Передать числа для вычисления |
| `/set: x = 100` | Сохранить переменную |
| `/undo` | Удалить последний обмен сообщениями |

После команды `/numbers` введите одно из действий: `+`, `-`, `*`, `/`, `gcd`, `avg`, `min`, `max`.

## Форматы чисел

| Формат | Пример | Результат |
|---|---|---|
| Запятые | `7, 9, 12` | [7, 9, 12] |
| Пробелы | `7 9 12` | [7, 9, 12] |
| Точка с запятой | `7;9;12` | [7, 9, 12] |
| Диапазон | `10..15` | [10, 11, 12, 13, 14, 15] |
| Смешанный | `5, 10..15, 20` | [5, 10, 11, 12, 13, 14, 15, 20] |

## Переменные

Бот автоматически сохраняет последний результат в переменную `ans`:
```
/numbers: ans, 2
*
```

Пользовательские переменные задаются через `/set: x = 100`.

## Дополнительно

- **История команд** — стрелки ↑ / ↓ для навигации по введённым командам.
- **Команда `/undo`** — удаляет последнее сообщение пользователя и ответ бота.
